为vim添加漂亮的 Solarized 配色方案,效果如下图。

Solarized 支持大部分编辑器及IDE,官方网站在这里。

vim 配置 Solarized

download

从github https://github.com/altercation/vim-colors-solarized 下载Vim-only 的仓库,即可完成vim的配置。

主仓库在 https://github.com/altercation/solarized ,只配置vim的话可以忽略。

install

$ cd vim-colors-solarized/colors
$ mv solarized.vim ~/.vim/colors/

修改.vimrc文件

syntax enable
set background=dark
colorscheme solarized

background 支持 dark 和 light 两种配色方案

mac terminal 支持

mac terminal 需要在 colorschem solarized 前添加一行配置

let g:solarized_termcolors=256

官方的描述是

If you do use the custom terminal colors, solarized.vim should work out of the box for you. If you are using a terminal emulator that supports 256 colors and don’t want to use the custom Solarized terminal colors, you will need to use the degraded 256 colorscheme.

vimrc 基础设置

分享一下vimrc的基础设置

syntax enable               "语法高亮"
set background=dark
let g:solarized_termcolors=256
colorscheme solarized

set nocompatible            "不兼容vi"
set nu                      "显示行数"
set expandtab               "用空格代替tab"
set shiftwidth=4            "缩进的字符个数" 
set tabstop=4               "一个tab是4个字符"
set cursorline              "为光标所在行加下划线"
set autoread                "文件在Vim之外修改过，自动重新读入"
set ignorecase              "检索时忽略大小"
set fileencodings=utf-8,gbk "使用utf-8或gbk打开文件"
set hls                     "检索时高亮显示匹配项"
set ruler                   "显示光标当前位置"
set laststatus=2            "总是显示状态栏"
set backspace=indent,eol,start
set nowrap                  "禁止折行"

"恢复文件关闭之前光标的位置"
if has("autocmd")
au BufReadPost * if line("'\"") > 1 && line("'\"") <= line("$") | exe "normal! g'\"" | endif
endif
